# Kanata Zed Extension

Zed editor extension adding support for [Kanata](https://github.com/jtroo/kanata/) configuration files.

## Features

- Syntax highlighting via Tree-sitter grammar
- Language server integration 

## Installation

Install from the Zed extension marketplace or clone this repository and load it as a local extension.
